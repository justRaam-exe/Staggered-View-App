# Anime Wallpaper App

Project Wallpaper App
Mata Kuliah Pemrograman Mobile menggunakan Java Native
project ini saya buat hampir mirip dengan Apk Pinterest
dengan menggunakan Staggered View Layout, dengan fitur :
- Home : fitur yang menampilkan deretan gambar
- Search : fitur yang hampir mirip dengan Home dan memiliki Search bar untuk mencari gambar yang diinginkan
- Profile : fitur yang berisi About page, About Developer, Theme Setting, Send Feedback, Privacy & policy, Version App
